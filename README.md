# Mi Tercer Addon de Minecraft

Este es mi tercer addon de Minecraft con recursos y comportamientos personalizados.

## Estructura del Proyecto

```
Mi_tercer_addon/
├── resources/
│   ├── assets/
│   │   └── modid/
│   │       ├── textures/
│   │       ├── models/
│   │       └── lang/
│   └── data/
│       └── modid/
│           ├── recipes/
│           └── loot_tables/
├── behaviors/
│   ├── scripts/
│   ├── events/
│   └── conditions/
└── pack.mcmeta
```

## Instalación

1. Descarga el addon
2. Coloca la carpeta en tu carpeta de addons de Minecraft
3. ¡Disfruta!
