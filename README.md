# Mi Primer Mod de Minecraft

Este es mi primer mod de Minecraft con recursos y comportamientos personalizados.

## Estructura del Proyecto

```
Mi_primer_mod/
├── resources/
│   ├── assets/
│   │   └── modid/
│   │       ├── textures/
│   │       ├── models/
│   │       └── lang/
│   └── data/
│       └── modid/
│           ├── recipes/
│           └── loot_tables/
├── behaviors/
│   ├── scripts/
│   ├── events/
│   └── conditions/
└── pack.mcmeta
```

## Instalación

1. Descarga el mod
2. Coloca la carpeta en tu carpeta de mods de Minecraft
3. ¡Disfruta!
